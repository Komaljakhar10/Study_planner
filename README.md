# Study_planner
Developed an AI-based student help platform integrating Hugging Face API to provide responses for academic queries through a simple web interface
An intelligent study planner and quiz generation system built using FastAPI, SQLite, HTML/CSS/JavaScript, and NLP techniques.
The project helps students organize study schedules, generate quizzes automatically from text, track quiz performance, and improve productivity through smart planning.

**Features**
Smart Study Timetable Generation
AI-based MCQ Quiz Generator
Quiz Performance Tracking
SQLite Database Integration
FastAPI Backend APIs
Interactive Frontend Dashboard
NLP-based Question Generation using NLTK
Optional Hugging Face API Integration
